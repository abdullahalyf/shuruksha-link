# Shuruksha Link

AI-assisted rural triage system.

Frontend:
- React
- Vite
- Tailwind

Backend:
- Express.js
- Gemini API

Deployment:
Frontend:
https://shuruksha-link.vercel.app

Backend:
https://shuruksha-link-api.onrender.com

Completed Features:
- Patient Information
- Vitals Intake
- OCR
- Voice Capture
- Gemini Triage
- Emergency Override
- Referral Engine
- PDF Export
- Case History
- Bengali to English PDF Translation
- Branding and Logo
